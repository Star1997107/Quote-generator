# Генератор случайных цитат

Простое GUI‑приложение для генерации случайных цитат с сохранением истории.

Автор - Лисун Мария

## Функциональность

* генерация случайных цитат из JSON‑файла;
* отображение текущей цитаты;
* история последних 10 цитат;
* автоматическое сохранение истории.
